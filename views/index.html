Example Usage:

To Translate Text:
  • Submit the 'Translate Text' form above with the desired text and translation selected.
  • Alternatively, send a POST request to /api/translate with a body consisting of url encoded fields of text (text to be translated) and locale('american-to-british' or 'british-to-american', the desired translation type).

Example Output:

On submitting a text that requires translation, the response body is JSON with the following form:

# Submitted fields:
{
  • "text": "I had a bicky then went to the chippy.",
  • "locale": "british-to-american",
}

# Response Body:
{
  • "translation": "I had a <span class="highlight">cookie</span> then went to the <span class="highlight">fish-and-chip shop</span>.",
  • "plaintext": "I had a cookie then went to the fish-and-chip shop."
  • "text": "I had a bicky then went to the chippy."
}

On submitting a text that does not require translation, the response body is JSON with the following form:

# Submitted fields:
{
  • "text": "These Mangoes are delicious!",
  • "locale": "american-to-british",
}

# Response Body:
{
  • "translation": "Everything looks good to me!",
  • "plaintext": "These Mangoes are delicious!"
  • "text": "These Mangoes are delicious!"
}

If the text or locale fields are missing, empty or invalid, a JSON with an 'error' property, with corresponding value being a string explaining the error is returned.
